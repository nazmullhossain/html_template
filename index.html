About Photo Collage Maker & Editor App

P

Photo Collage Maker & Editor (Flutter) helps users design beautiful collages and edit photos with professional tools. Choose layouts and templates, add multiple images, customize backgrounds, and fine-tune with brightness, contrast, saturation, hue, and blur.

Enhance creativity with stickers, emojis, frames, and advanced text controls—curve text, shadows, opacity, and rich formatting. Save, download, or share instantly. Works on both Android and iOS.

Key Highlights

  • Cross-platform app built with Flutter (works on Android & iOS)
  • Create stunning photo collages with customizable layouts & templates
  • Powerful photo editing tools: crop, resize, rotate, flip, blur
  • 40+ stylish fonts, neon text, curve text, colors, and shadows
  • Apply filters, frames, stickers, emojis, and tattoos
  • Save photos locally or share instantly on social media
  • Integrated with Firebase, AdMob, and push notifications

Flutter Setup Guide

Prerequisites

  1. Download and install Flutter SDK
  2. Install Android Studio or Xcode (for iOS development)
  3. Install VS Code or Android Studio with Flutter plugin

Setting Up Flutter

  1. Extract the Flutter SDK zip file to your desired location
  2. Add Flutter to your PATH: 
    export PATH="$PATH:`pwd`/flutter/bin"
  3. Run flutter doctor to check for dependencies: 
    flutter doctor
  4. Install any missing dependencies as indicated by flutter doctor

Configuring an Editor

For VS Code:

  1. Install the Flutter extension from the marketplace
  2. Install the Dart extension
  3. Configure the Flutter SDK path in settings

Installation

From Source

  1. Clone the repository: 
    git clone https://github.com/yourusername/docscan_flutter.git
  2. Navigate to the project directory: 
    cd docscan_flutter
  3. Get dependencies: 
    flutter pub get
  4. Run the app: 
    flutter run

From Play Store/App Store

Download the app from:

  • Google Play Store
  • Apple App Store

How to Use

Create a Collage

  1. Open the Collage Maker – Photo Editor app
  2. Tap the “Create Collage” button
  3. Select multiple photos from your gallery
  4. Choose a layout or template for your collage
  5. Adjust image positions, crop, or resize as needed
  6. Customize with backgrounds, stickers, text, and filters
  7. Save your collage or share directly on social media

Edit a Photo

  1. Tap on “Edit Photo” from the home screen
  2. Select a single photo from your gallery
  3. Apply filters, frames, and enhancements (brightness, contrast, saturation, hue, blur)
  4. Add text with 40+ fonts, neon effects, shadows, and curve styles
  5. Insert stickers, emojis, or tattoos for a creative touch
  6. Save your edited photo or share instantly

Save & Share

  • Save your collage or photo to your device
  • Share instantly on social platforms like Facebook, Instagram, or WhatsApp
  • Download high-quality images in JPG/PNG format

App Customization

Changing Package Name

To change the package name of your Flutter app:

For Android:

  1. Open android/app/build.gradle
  2. Change the applicationId in the defaultConfig section
  3. Rename the package directory structure in android/app/src/main/kotlin (or java) to match your new package name
  4. Update the package name in AndroidManifest.xml

For iOS:

  1. Open the iOS project in Xcode (ios/Runner.xcworkspace)
  2. Select the Runner project in the project navigator
  3. Change the bundle identifier in the General tab
  4. Update the bundle identifier in Info.plist

Changing App Name

  1. For Android: Update android/app/src/main/AndroidManifest.xml (android:label)
  2. For iOS: Update ios/Runner/Info.plist (CFBundleDisplayName)

Changing App Icon

Use the flutter_launcher_icons package:

  1. Add to pubspec.yaml: 
    dev_dependencies:
  flutter_launcher_icons: "^0.13.1"
            
flutter_launcher_icons:
  android: true
  ios: true
  image_path: "assets/icon/icon.png"
  2. Place your icon image at the specified path
  3. Run: flutter pub run flutter_launcher_icons:main

Google AdMob Integration

Setup AdMob Account

  1. Create an account at admob.google.com
  2. Add your app to AdMob dashboard
  3. Create ad units for banner, interstitial, and rewarded ads

Adding AdMob to Flutter

  1. Add dependencies to pubspec.yaml: 
    dependencies:
  google_mobile_ads: ^4.0.0
  2. Initialize AdMob in your main.dart: 
    import 'package:google_mobile_ads/google_mobile_ads.dart';

void main() {
  WidgetsFlutterBinding.ensureInitialized();
  MobileAds.instance.initialize();
  runApp(MyApp());
}

Implementing Banner Ads

BannerAd(
  adUnitId: 'your-ad-unit-id',
  size: AdSize.banner,
  request: AdRequest(),
  listener: BannerAdListener(),
).load();

Implementing Interstitial Ads

InterstitialAd.load(
  adUnitId: 'your-interstitial-ad-unit-id',
  request: AdRequest(),
  adLoadCallback: InterstitialAdLoadCallback(
    onAdLoaded: (InterstitialAd ad) {
      // Store the ad to show later
      this.interstitialAd = ad;
    },
    onAdFailedToLoad: (LoadAdError error) {
      print('InterstitialAd failed to load: $error');
    },
  ),
);
